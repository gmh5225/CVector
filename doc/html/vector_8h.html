int  num,

)

Create and initialize vector with num elements of vals.

If vals is NULL, capacity is set to num + VEC_I_START_SZ. Size is set to num in the first place, 0 otherwise. elem_sz is the size of the type you want to store ( ie sizeof(T) where T is your type ).

vector_s* vec_s ( int  sz  ) 

Create a new vector_s with size sz.

Capacity is set to sz + VEC_S_START_SZ. Note: vector_s does not copy pointers passed in but duplicates the strings they point to (using mystrdup()) so you don't have to worry about freeing or changing the contents of variables that you've pushed or inserted; it won't affect the vector.
